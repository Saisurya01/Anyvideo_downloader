# AnyVideo Downloader

A simple web application that allows users to download videos from supported platforms by providing a video URL. The app provides a clean interface where users can paste a link and download the video directly.

⚠️ **Important:** This project currently does **not work on free hosting platforms like GitHub Pages** because video downloading requires server-side processing.

---

## Overview

AnyVideo Downloader is a web-based tool designed to simplify downloading videos from the internet. Users only need to paste a video link, and the application processes the request and returns downloadable video formats.

This project was built using **vibe coding**, where most of the development was done through AI-assisted coding and rapid experimentation.

---

## Features

- Simple and clean user interface
- Paste video URL and download
- Supports multiple video sources
- Fast link processing
- Beginner-friendly project structure

## Why It Doesn't Work on GitHub Pages

GitHub Pages only supports **static websites** (HTML, CSS, JS).  
Video downloading requires **backend processing**, which means:

- Handling video streams
- Processing URLs
- Fetching video files from servers

These tasks require a **server environment**, which GitHub Pages cannot provide.
